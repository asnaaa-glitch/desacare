❓ Panduan Penggunaan DesaCare

📌 Apa itu DesaCare?

DesaCare adalah sistem digital untuk melaporkan masalah desa dan melihat transparansi dana desa secara online. Aplikasi ini dibuat agar masyarakat bisa ikut berpartisipasi dalam pembangunan desa.

🧭 Navigasi (Sidebar)

Sidebar adalah menu utama di sebelah kiri layar. Dari sini kamu bisa pindah halaman tanpa reload.

Cara pakai:

  1. Klik menu di sidebar (Home, Lapor, Dashboard, dll)
  2. Halaman akan langsung berubah
  3. Menu aktif akan berwarna berbeda

🧾 Cara Melapor Masalah

  1. Pilih menu "Lapor"
  2. Isi nama, jenis masalah, alamat, dan deskripsi
  3. Upload foto sebagai bukti
  4. Klik tombol kirim
  5. Laporan akan masuk ke sistem dengan status Pending

📊 Dashboard

Dashboard adalah halaman ringkasan data. Di sini kamu bisa melihat jumlah laporan dan statusnya secara cepat.

Cara pakai:

  • Buka menu Dashboard
  • Lihat jumlah laporan masuk
  • Lihat status (Pending, Diproses, Selesai)

📍 Status Laporan

  • 🟡 Pending → laporan baru masuk
  • 🔵 Diproses → sedang dikerjakan admin
  • 🟢 Selesai → sudah selesai ditangani

💰 Transparansi Dana

Halaman ini menampilkan informasi penggunaan dana desa agar masyarakat bisa melihat secara terbuka.

Cara pakai:

  • Buka menu Dana
  • Lihat daftar transaksi
  • Cek jumlah dan bukti dana

🔔 Notifikasi (Toast)

Notifikasi kecil yang muncul di pojok layar untuk memberi informasi seperti: "Laporan terkirim" atau "Status diperbarui".

🔐 Login Admin

Digunakan untuk masuk ke halaman admin agar bisa mengelola laporan dan data desa.

Cara pakai:

  1. Buka menu Login
  2. Masukkan username & password
  3. Klik login
  4. Jika benar, masuk ke dashboard admin
